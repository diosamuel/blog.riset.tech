AssemblyAI is an applied AI company that provides advanced speech-to-text, voice agent, and audio intelligence APIs. International hackathon with 75 participants across three categories.

- Developed a full-stack web system, including a web client for user reporting and a separate web client for a dashboard monitoring system used to track SOS distress reports.
- Integrated an AI-based LLM service, AssemblyAI, into the entire system.
- Leveraged LeMUR to build LLM-based features on top of voice data, enabling advanced voice analysis and automated response capabilities.
- Won the LLM-powered application category and a $1,000 prize.
